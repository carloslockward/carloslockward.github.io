Deep Q Learning for Cryptocurrency Trading

Project information

  • Date: May 7, 2021
    • PDF

This project consists of an implementation of a Deep Q Learning trading model that can determine whether to buy, sell or hold, based on some past price information and the analysis of the contribution that technical indicators have in the performance of the aforementioned model. The model performed similarly to previous work, and we showed that using technical indicators can significantly improve performance.
